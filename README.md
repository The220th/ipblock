# Что это?

`ipblock` - небольшой скрипт, который позволит работать с IPv4, переводя их в разные системы счисления, и который позволит вычислить сеть по маске.

# Использование

Есть 2 режима работы:

1. Перевод чисел из двоичной в десятичную и наоборот. 

1. Основной. 

## Первый режим

Синтаксис: `python ipblock.py число_системаСчисления`. Система счисления должна быть либо `2`, либо `10`.

``` bash
> python ipblock.py 123_10 # Выведет 01111011

> python ipblock.py 01111011_2 # Выведет 123

> python ipblock.py 1111011_2 # Выведет 123
```

## Второй режим

Синтаксис: `python ipblock.py ip маска`, где `ip` и `маску` можно вводить комбинированным способом, например:

- `192.168.25.13`

- `192.168.25.00001101`

- `11000000.10101000.25.00001101`

- `11000000.10101000.00011001.00001101`

Маску можно вводить с помощью префикса. Например:

- `255.255.255.0` и `/24` одно и то же

- `255.255.255.224` и `/27` одно и то же

- `255.248.0.0` и `/13` одно и то же

- `255.255.255.255` и `/32` одно и то же

- `0.0.0.0` и `/0` одно и то же
